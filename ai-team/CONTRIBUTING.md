# Contributing to AI Team MCP

Thanks for your interest in contributing!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/ai-team-mcp.git`
3. **Create a branch**: `git checkout -b feature/my-feature`
4. **Make your changes**
5. **Test**: `python3 aiteam.py status`
6. **Commit**: `git commit -m "Add my feature"`
7. **Push**: `git push origin feature/my-feature`
8. **Open a Pull Request**

## Ideas for Contributions

- Add new AI providers (Mistral, Llama, Groq, etc.)
- Improve agent prompts for better results
- Add streaming responses
- Build a web dashboard
- Add conversation memory/history
- Write tests
- Improve documentation
- Add VS Code / JetBrains extensions
- Cost/token tracking

## Code Style

- Python 3.10+
- Keep it simple — no over-engineering
- Each agent in its own file under `agents/`
- Follow existing patterns

## Reporting Bugs

Open an issue with:
- What you expected
- What happened
- Steps to reproduce
- Your OS and Python version

## Questions?

Open a Discussion on GitHub.
